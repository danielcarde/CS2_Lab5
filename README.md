## Overview
Lab 5 is a smaller CS2 exercise that keeps the implementation in one Java file. It is intended for focused practice without the additional structure of a multi-class project.
The lab fits into the larger sequence between linked structure assignments and later search/tree work.

## Why This Project
Short exercises are useful for practicing a specific concept without extra project overhead. This lab provides that kind of targeted reinforcement.

## Usage
Compile: Compile using javac Lab5.java.
Run: Execute with java Lab5.
Review the source comments and console output for the lab behavior.

## Installation
Ensure you have the Java Development Kit (JDK) installed on your system.
Open a terminal in the lab folder before compiling so the program can find any required local text files.

## Architecture
File Structure: The project consists of Lab5.java.
Components: The Java file contains the program logic and entry point.
